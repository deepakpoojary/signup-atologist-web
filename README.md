# Next.js Project

A production-ready Next.js application built with React, App Router, and optimized for modern web development.

## Features
- Next.js App Router
- Server and Client Components
- API Routes 
- Built-in SEO support
- Image Optimization
- Fast Refresh for development
- Environment variable support

## Tech Stack
- Next.js
- React
- JS
- SCSS

## Getting Started

### Prerequisites
- Node.js 18+  
- npm, yarn, pnpm, or bun

### Install Dependencies
```bash
npm install 
```
### Run Development Server
```bash
npm run dev
```

The app runs at:

http://localhost:3000

Build for Production
```bash
npm run build
npm run start
```
